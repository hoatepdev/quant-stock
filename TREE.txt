Database Tables:
================
1. stock_info              - Stock metadata
2. daily_price            - OHLCV data (TimescaleDB)
3. financial_statement    - Quarterly/annual financials
4. financial_ratio        - Calculated ratios
5. corporate_action       - Splits, dividends
6. factor                 - Investment factors (TimescaleDB)
7. market_index          - Index data (TimescaleDB)
8. data_quality_log      - Data validation logs

API Endpoints:
==============
POST   /api/v1/screen                   - Screen stocks
GET    /api/v1/tickers                  - List tickers
GET    /api/v1/factors/{ticker}         - Get stock factors
GET    /api/v1/factors/{ticker}/history - Factor history
GET    /api/v1/factors/available        - List factors
GET    /api/v1/health                   - Health check
GET    /api/v1/ready                    - Readiness check
GET    /docs                            - Swagger UI
GET    /redoc                           - ReDoc UI
